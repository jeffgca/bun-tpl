To create a new repo with this project template using the gh cli:

```bash
gh repo create $PROJECT_NAME --template jeffgca/bun-tpl --private --clone'
```

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

To run tests:

```bash
bun test
```

To compile to a standalone binary:

```bash
bun cc
```
